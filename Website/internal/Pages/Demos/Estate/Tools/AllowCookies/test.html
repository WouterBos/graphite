  • Description

    De bezoeker krijgt de donkere balk te zien zodra het JavaScript bestand is ingeladen.

    Nadat de gebruiker een keuze heeft gemaakt, wordt de pagina opnieuw ingeladen (zodat server-side code cookies kunnen uitlezen).

    Bij het herladen van de pagina wordt "#cPrompt=set" toegevoegd aan de URI. Als dat in een adres zit, wordt de keuzebalk zichtbaar, ook al heeft de bezoeker al een keuze gemaakt. Op die manier heeft de gebruiker nog kans om de instellingen aan te passen in geval van een "post decisional regret".

    Als het script is ingeladen, kan je door middel van cPrompt.cookiesAllowed() checken of de website cookies mag gebruiken (true) of niet (false).

    Het script is gebaseerd op cPrompt, maar is danig aangepast. Het is dus niet mogelijk om updates te installeren.
